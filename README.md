# 淮安大富翁游戏

一个基于Vue3 + Node.js + MySQL的多人在线大富翁游戏，融合淮安地理文化特色和金融常识。

## 技术栈

- **前端**: Vue 3 + Vite + TypeScript + Element Plus
- **后端**: Node.js + Express + Socket.io
- **数据库**: MySQL
- **实时通信**: WebSocket (Socket.io)

## 功能特色

- 🎮 最多支持6人联机游玩
- 🗺️ 淮安特色地图设计，40个特色地标
- 💰 完整的金融系统：拍卖、抵押、股票交易
- 🏘️ 地产购买、建设、租金收取
- 🎯 多种游戏卡牌和随机事件
- 💬 实时聊天系统
- 🏆 排行榜和成就系统
- 🎴 独特的运河宝藏系统
- 📈 动态股票市场
- 🔨 实时拍卖系统

## 快速开始

### 环境要求

- Node.js >= 16.0.0
- MySQL >= 8.0
- npm >= 8.0.0

### 安装依赖

```bash
npm run install:all
```

### 配置数据库

1. 创建MySQL数据库
2. 导入数据库脚本: `database/schema.sql`
3. 配置数据库连接: `server/config/database.js`

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 部署到openEuler服务器

```bash
npm run build
npm start
```

## 🎮 完整游戏玩法说明

### 游戏目标
成为淮安最富有的玩家！通过买卖地产、建设房屋、收取租金、投资股票等方式积累财富，成为最后的赢家。

### 游戏界面
- **游戏棋盘**: 40个格子组成的淮安特色地图，包含各种地标和特殊位置
- **玩家信息面板**: 显示所有玩家的资金、地产、总资产等信息
- **聊天系统**: 实时与其他玩家交流
- **操作按钮**: 掷骰子、确认事件等操作

### 基础游戏规则

#### 🎲 回合制玩法
1. **掷骰子**: 轮到你时点击"掷骰子"按钮
2. **移动**: 根据骰子点数在棋盘上移动
3. **处理事件**: 根据落在的位置处理相应事件
4. **结束回合**: 完成所有操作后结束回合

#### 💰 起始设置
- 每位玩家起始资金：**1500元**
- 起始位置：**起点**（淮安市政府广场）
- 经过或停留在起点获得：**200元**

#### 🏘️ 地产系统

**地产分类**：
- **棕色组**: 河下古镇、淮安驸马巷（古镇文化区）
- **浅蓝组**: 周恩来故居、吴承恩故居、里运河（伟人故里）
- **粉红组**: 淮安府署、镇淮楼、中国漕运博物馆（古建遗址区）
- **橙色组**: 中国淮扬菜博物馆、楚秀园、里运河文化长廊（淮扬美食区）
- **红色组**: 周恩来纪念馆、淮安大剧院、淮安市体育中心（现代文化区）
- **黄色组**: 白马湖旅游景区、淮安西游乐园、古淮河文化生态景区（生态旅游区）
- **绿色组**: 淮安国际食品博览中心、江苏食品药品职业技术学院、淮阴博物馆（教育科研区）
- **深蓝组**: 淮安国际商城步行街、江苏银行淮安分行（商业金融区）
- **交通枢纽**: 淮安东站、汽车站、新长铁路、宁淮高速
- **公用事业**: 淮阴电力、自来水公司

**地产操作**：
- **购买**: 停留在无主地产时可以购买
- **建设**: 拥有整个颜色组后可以建造房屋（每处地产最多4栋）
- **租金**: 其他玩家停留在你的地产时需要支付租金
- **垄断效果**: 拥有同色组全部地产时获得特殊效果

#### 🎯 特殊位置

**🎲 机会**: 抽取机会卡牌，可能获得金钱、移动、特殊效果等
**🏛️ 公共服务**: 抽取公共服务卡牌，涉及社区建设、税收、福利等
**🏺 运河码头**: 抽取运河宝藏卡，获得特殊奖励和buff效果
**💸 税收**: 所得税（200元）、奢侈税（750元）
**🔒 监狱**: 暂停行动，需要掷出对子、等待3回合或支付500元保释
**🚨 去监狱**: 直接前往监狱，不经过起点

### 🏦 金融系统

#### 🔨 拍卖系统
- **触发条件**: 玩家拒绝购买地产时自动进入拍卖
- **拍卖规则**: 60秒倒计时，所有玩家可以参与竞拍
- **出价规则**: 每次出价必须高于当前最高价
- **结算**: 时间结束后最高出价者获得地产

#### 🏦 抵押系统
- **抵押条件**: 地产无房屋建筑时可以抵押
- **抵押价值**: 地产价格的50%
- **利息**: 年利率10%，按天计算
- **赎回**: 支付本金+利息可以赎回地产

#### 📈 股票交易系统
**可投资股票**：
- **淮安食品集团** (HUAI_FOOD): 起始价格100元
- **淮安科技** (HUAI_TECH): 起始价格150元  
- **淮安文旅** (HUAI_TOURISM): 起始价格80元
- **淮安银行** (HUAI_BANK): 起始价格120元

**交易规则**：
- **买入**: 消耗现金购买股票
- **卖出**: 出售持有股票获得现金
- **价格波动**: 受供需关系和随机市场因素影响
- **最低价格**: 每股最低10元

### 🎴 卡牌系统

#### 🎲 机会卡牌
- 政府扶持、市场机遇、投资收益
- 前往特定位置、获得金钱奖励
- 房产升值、交通罚款等

#### 🏛️ 公共服务卡牌
- 慈善捐款、纳税优惠、社区建设
- 医疗费用、教育投资、环保奖励
- 每位玩家支付或获得特定金额

#### 🏺 运河宝藏卡牌
- **运河宝藏**: 获得2000元巨额奖励
- **航运特权**: 获得运河船只，解锁特殊能力
- **运河加持**: 运河相关地产租金提升20%（5回合）
- **商贸兴盛**: 下次收取租金时翻倍
- **股市内幕**: 股票投资收益翻倍（3回合）
- **免疫符**: 免疫下次监狱或罚款
- **强制交易令**: 与任意玩家强制交易一次

### 🎯 Buff系统

**租金相关**：
- **doubleRent**: 下次收取租金翻倍
- **rentBonus**: 租金收入增加指定百分比
- **canalBonus**: 运河相关地产租金提升

**建设相关**：
- **buildDiscount**: 建造房屋成本减半

**税收相关**：
- **taxRelief**: 下次税费减半
- **immunity**: 免疫罚款和监狱

**特殊能力**：
- **foresight**: 预知下次骰子结果
- **forceTrade**: 强制交易能力
- **stockBonus**: 股票投资收益翻倍

### 🏆 垄断效果

- **古镇文化区**: 租金+15%
- **伟人故里**: 每回合+20元文化津贴
- **古建遗址区**: 租金+25%
- **淮扬美食区**: 每回合随机获得50-200元美食收入
- **现代文化区**: 租金+30%
- **生态旅游区**: 每3回合+300元生态奖励
- **教育科研区**: 建造成本-25%，租金+35%
- **商业金融区**: 租金+50%，可进行高级金融操作

### 🎮 游戏策略

#### 初期策略
1. **积极购买地产**: 尽快建立地产组合
2. **关注同色组**: 优先完成垄断获得特殊效果
3. **合理使用现金**: 保持足够现金应对意外支出

#### 中期策略
1. **建设房屋**: 垄断后积极建造房屋提升租金
2. **关注拍卖**: 参与拍卖获得关键地产
3. **投资股票**: 适当投资股票分散风险

#### 后期策略
1. **抵押融资**: 利用抵押系统获得发展资金
2. **高级金融**: 充分利用拍卖和股票系统
3. **战略交易**: 通过交易完成关键垄断

### 🏁 游戏结束条件

- **胜利条件**: 只剩一名玩家未破产
- **破产条件**: 玩家现金为0且无法通过出售资产偿还债务
- **时间限制**: 超过200回合按总资产排名

### 💡 游戏技巧

1. **优先策略**: 棕色和浅蓝色地产投资回报率高
2. **现金管理**: 保持30%以上现金比例应对突发情况
3. **交易时机**: 在对手接近破产时进行有利交易
4. **buff利用**: 合理使用各种buff效果最大化收益
5. **股票时机**: 低价买入，高价卖出，关注市场波动
